# 制約と今後の拡張候補

## 現在の制約

### オフライン

- 初回アクセス前の完全オフライン利用はできない。
- Service Worker登録は本番環境で有効になる。
- ブラウザのキャッシュやサイトデータ削除で保存済みデータが失われる可能性がある。

### 育成シミュレータ

- Pokemon Champions向けの能力ポイント制を前提にしている。
- 本編ポケモンの努力値仕様とは異なる。
- 保存済み育成案はブラウザ内にのみ保存される。

### バトルチーム

- チームは1〜6体。
- 同じポケモンの重複は不可。
- 同じ持ち物の重複は不可。
- チーム共有やインポート/エクスポートは未実装。

### ダメージ計算

- ダメージ計算はPokemon Champions向けルールセットを前提にしている。
- 一部の特性/持ち物は手動条件のオン/オフで扱う。
- すべての対戦中効果を完全再現しているわけではない。

### 対戦シミュレータ

現在は簡易版である。

実装済み:

- 2チーム選択
- 対戦開始
- 技選択
- 交代選択
- ターン実行
- 交代を技より先に処理
- 素早さ順で技処理
- ダメージ反映
- ひんし判定
- 控えの自動選出
- 勝敗ログ

未実装または簡易扱い:

- 命中判定
- PP
- 優先度
- 急所
- 乱数ダメージの選択
- 状態異常
- ランク変化
- 天候/フィールド操作
- 場に残る効果
- 特性の完全自動発動
- 持ち物の完全自動発動
- 対戦結果の保存

## 今後の拡張候補

- 条件付き特性/持ち物のUI改善
- 場の状態のプリセット化
- 複数技の比較

### 対戦シミュレータ

- PP管理
- 命中判定
- 優先度
- 状態異常
- ランク変化
- 天候/フィールド効果
- 特性/持ち物の発動ログ
- 手動で次に出すポケモンを選ぶフロー
- 対戦ログの保存/再生
