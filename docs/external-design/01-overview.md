# 概要

PokemonLab は、ポケモンの検索、タイプ相性確認、育成案作成、バトルチーム編成、ダメージ計算、対戦シミュレーションを行うためのPWAである。

アプリはブラウザ上で動作し、一度必要なデータを読み込んだ後はオフライン利用できることを前提にしている。

## 目的

- ポケモン関連データを素早く検索する。
- タイプ相性を確認し、クイズで学習する。
- Pokemon Champions向けの育成案を作成・保存する。
- 保存済み育成案からバトルチームを作成する。
- チームや個体を使ってダメージ計算を行う。
- 1人でPlayer 1とPlayer 2を操作して、簡易的な対戦を試す。

## 利用者

- Pokemon Champions向けの育成案を考えたいユーザー
- チーム単位でダメージや対戦展開を確認したいユーザー
- オフラインでも手元の保存データを使って検討したいユーザー

## アプリの基本方針

- 主要機能はブラウザ内で完結する。
- カタログデータは同梱されたSQLiteデータから読み込む。
- ユーザーが作成した育成案やバトルチームはブラウザ内SQLiteに保存する。
- 外部APIへの実行時依存を持たない。
- スマートフォンでも主要操作ができるレイアウトにする。

## 共通UI

全画面で以下の共通UIを持つ。

| UI | 内容 |
|---|---|
| ヘッダー | 主要機能へのナビゲーションを表示する。 |
| PokemonLabリンク | ホームへ戻る。 |
| タイプ相性表ボタン | どの画面からでもタイプ相性表を開く。 |
| メニューボタン | モバイル表示でナビゲーションを開閉する。 |
| 戻るボタン | 前のページへ戻る。 |

## オフライン前提

PokemonLab はPWAとして動作する。
本番ビルドではService Workerを登録し、静的ファイルやデータベース資産をキャッシュする。

ただし、初回アクセス前に完全なオフライン利用はできない。
少なくとも一度、アプリ本体と必要なデータを読み込む必要がある。
