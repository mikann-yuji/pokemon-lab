# PokemonLab 外部設計書

このディレクトリは、PokemonLab の外部設計をまとめる。
外部設計では、ソースコードの実装詳細ではなく、利用者から見える画面、機能、画面遷移、データの流れ、制約を扱う。

## 対象範囲

- ホーム
- ポケモン検索/詳細
- タイプ相性表
- タイプ相性クイズ
- 育成シミュレータ
- 保存済み育成案一覧
- バトルチーム編成
- ダメージ計算
- 対戦シミュレータ
- SQLite診断

## この文書で扱わないこと

- Reactコンポーネントの詳細な責務分解
- TypeScript型、関数、DBクエリの詳細仕様
- ダメージ計算や対戦処理の内部アルゴリズム詳細
- ディレクトリ構成とモジュール依存の完全な説明

これらは内部設計書で扱う。
