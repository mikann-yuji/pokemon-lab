## 機能詳細

### ポケモン検索

- キーワードでポケモンを検索する。
- 通常のポケモン検索と、育成対象に絞った検索がある。
- 検索結果から詳細画面または育成シミュレータへ遷移する。

### 育成シミュレータ

- ポケモン単位で育成案を編集する。
- 性格、能力ポイント、持ち物、特性、技を設定する。
- 実数値を表示する。
- 保存済み育成案がある場合は復元できる。
- 同じ内容の育成案が既にある場合は既存データを利用する。

### バトルチーム編成

- 保存済み育成案から1〜6体のチームを作成する。
- 同じポケモンや同じ持ち物の重複は許可しない。
- 作成済みチームは編集できる。

### ダメージ計算

- 攻撃側と防御側のポケモンを選択する。
- 保存済みバトルチームからポケモンを選択できる。
- 最近使ったポケモンを復元できる。
- 選択中ポケモンの画像、名前、種族値を表示する。
- ポケモンのサマリーから育成シミュレータ詳細へ遷移できる。
- 技、特性、持ち物、場条件を反映してダメージ範囲を表示する。
- 素早さ比較モーダルで簡易的な素早さ確認ができる。

### 対戦シミュレータ

- 保存済みバトルチームをPlayer 1とPlayer 2にそれぞれ選択する。
- 1人のユーザーが両プレイヤーの行動を選ぶ。
- 対戦開始後、Player 1とPlayer 2の行動選択はタブで切り替える。
- 技は場に出ているポケモンの技リストから選択する。
- 交代は交代ボタンからモーダルを開き、控えポケモンを選択する。
- 両方の行動が選択されるとターン実行できる。
- 交代は技より先に処理する。
- 技は素早さ順で処理する。
- ダメージは簡易的に平均ダメージを採用してHPへ反映する。
- HPが0になった場合はひんし扱いにする。
- 控えがいる場合は自動で次のポケモンを出す。
- 全員ひんしになったプレイヤーが出た場合、勝敗ログを表示する。
- 対戦画面は、場の2体、対戦ログ、行動選択、ターン実行ボタンを縦に並べたコンパクトな構成にする。
