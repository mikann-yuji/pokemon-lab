# 画面一覧と画面ごとの機能

## 画面一覧

| パス | 画面名 | 目的 |
|---|---|---|
| `/` | ホーム | 主要機能への入口を表示する。 |
| `/pokemon` | ポケモン検索 | ポケモンを検索する。 |
| `/pokemon/[id]` | ポケモン詳細 | ポケモン詳細を表示する。 |
| `/training` | 育成ポケモン選択 | 育成対象ポケモンを検索する。 |
| `/training/[id]` | 育成シミュレータ詳細 | 指定ポケモンの育成案を作成・編集する。 |
| `/training-builds` | 保存済み育成案一覧 | 保存済み育成案を確認する。 |
| `/battle-team` | バトルチーム編成 | 保存済み育成案からチームを作成する。 |
| `/battle-team/new` | 新規バトルチーム作成 | 新しいチームを作成する。 |
| `/battle-team/[id]` | バトルチーム編集 | 既存チームを編集する。 |
| `/damage-calculator` | ダメージ計算 | 選択したポケモン同士のダメージを計算する。 |
| `/battle-simulator` | 対戦シミュレータ | 2チームで1人回し対戦を行う。 |
| `/quiz` | タイプ相性クイズ | タイプ相性のクイズに回答する。 |
| `/sqlite-diagnostics` | SQLite診断 | SQLite WASMとDB接続を確認する。 |

## `/` ホーム

### 目的

PokemonLabの主要機能への入口を提供する。

### 主な表示

- アプリ名
- PWA/オフライン対応を示すステータス
- 主要機能カード

### 主な遷移

- ポケモン検索
- ダメージ計算
- 育成シミュレータ
- バトルチーム編成
- 対戦シミュレータ
- タイプ相性クイズ

## `/pokemon` ポケモン検索

### 目的

カタログDBに登録されたポケモンを検索する。

### 主な操作

- キーワード入力
- 検索結果の選択

### 主な遷移

- `/pokemon/[id]`

## `/pokemon/[id]` ポケモン詳細

### 目的

指定ポケモンの詳細情報を表示する。

### 主な表示

- 名前
- 画像
- タイプ
- 種族値
- 技や特性などの詳細情報

## `/training` 育成ポケモン選択

### 目的

育成シミュレータで編集するポケモンを選ぶ。

### 主な操作

- 育成対象ポケモンを検索
- 検索結果を選択

### 主な遷移

- `/training/[id]`
- `/training-builds`

## `/training/[id]` 育成シミュレータ詳細

### 目的

指定ポケモンの育成案を編集・保存する。

### 主な操作

- 性格選択
- 能力ポイント調整
- 特性選択
- 持ち物選択
- 技選択
- 育成案保存

### 主な表示

- 実数値
- 能力ポイント合計
- 種族値ランキング
- 保存状態

## `/training-builds` 保存済み育成案一覧

### 目的

保存済み育成案を一覧し、再編集やチーム編成に使う。

### 主な操作

- 育成案確認
- 育成案選択
- チーム編成への利用

## `/battle-team` バトルチーム編成

### 目的

保存済み育成案からバトルチームを作成する。

### 主な操作

- 育成案を選択
- チーム名を入力
- チーム保存

### 主な制約

- チームは1〜6体。
- 同じポケモンは重複不可。
- 同じ持ち物は重複不可。

## `/damage-calculator` ダメージ計算

### 目的

攻撃側、防御側、技、場条件からダメージを計算する。

### 主な操作

- 攻撃側ポケモン選択
- 防御側ポケモン選択
- 保存済みバトルチームから選択
- 技選択
- 特性・持ち物・能力補正の調整
- 天候・フィールド選択
- 素早さ比較

### 主な表示

- 攻撃側/防御側のポケモンサマリー
- 種族値ミニ表示
- ダメージ範囲
- 相手の残りHP割合
- KO目安

## `/battle-simulator` 対戦シミュレータ

### 目的

保存済みバトルチーム2つを使って、1人でPlayer 1とPlayer 2を操作する。

### 主な操作

- Player 1のバトルチーム選択
- Player 2のバトルチーム選択
- 対戦準備作成
- 対戦開始
- Player 1の技/交代選択
- Player 2の技/交代選択
- ターン実行

### 主な表示

- Player 1/2のチーム名
- 現在場に出ているポケモン
- HP
- 控えポケモン一覧
- 技ボタン
- 交代ボタン
- 対戦ログ
- 勝敗ログ

### 現在の処理範囲

- 交代は技より先に処理する。
- 技は素早さ順で処理する。
- ダメージは平均値を採用する。
- HP0でひんし。
- 控えがいる場合は自動で次のポケモンを出す。

## `/quiz` タイプ相性クイズ

### 目的

タイプ相性をクイズ形式で学習する。

### 主な操作

- 問題への回答
- スコア確認
- ミス確認

## `/sqlite-diagnostics` SQLite診断

### 目的

SQLite WASMとデータベースの読み込み状態を確認する。

### 主な表示

- SQLite WASM状態
- catalog DB状態
- user DB状態
- 診断ログ
